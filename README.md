# 四社合同の日本AI基盤モデル開発 ― 2026年4月19日

2026年1月に設立され、同年4月12日に公表された国家プロジェクト
**「日本AI基盤モデル開発株式会社」** の全貌を、モバイル特化UIで
深く解説する特設Webサイト。

- 中核出資：**ソフトバンク / NEC / 本田技研工業 / ソニーグループ**
- 追加出資：**日本製鉄・神戸製鋼・三菱UFJ・三井住友・みずほ**
- 技術連携：**Preferred Networks (PFN)**
- 目標：**2030年までに1兆パラメーター級のマルチモーダル基盤モデル**
- 支援：**NEDO P26013（最大1兆円／5年）＋GENIAC**
- 拠点：**北海道苫小牧（約70ha、300MW超）＋大阪堺（旧シャープ工場）**

---

## 🎯 プロジェクトの目的

提供された2本のTXT（本文レポート＋出典109件）を
徹底的に読み込み、その内容を**忠実に・構造化して・スマホで快適に読める形**で
Web化した。画像は使用せず、タイポグラフィ／グラデーション／
CSSアニメーションのみで高い表現力を実現している。

---

## ✅ 完成済みの機能

### 📱 モバイル特化UI
- 幅360～430pxを主戦場とした**モバイルファースト設計**
- 画面下の**ボトムナビ**（ホーム／年表／企業／出典）
- 右からスライドする**全目次ドロワー**（ハンバーガー起動）
- **リーディング進捗バー**（ページ上部に常駐）
- スクロール追従の**「トップへ戻る」ボタン**

### 🎨 デザインシステム
- ダークネイビー基調（`#0b0d14`）＋ゴールド／シアンのアクセント
- Noto Sans JP / Noto Serif JP / JetBrains Mono の3書体構成
- CSS変数（`--c-*`, `--sp-*`, `--fs-*`）による一貫した設計
- **60種類以上のコンポーネント**（カード、タイムライン、
  進捗バー、コールアウト、アコーディオン、テーブル等）
- **IntersectionObserver**による段階的フェードイン登場
- 数値**カウントアップアニメーション**

### 📖 コンテンツ
- **全15章の詳細章立てページ**（各章 約250〜400行規模）
- 完全年表ページ（2026年1月〜2030年）
- 出資企業一覧ページ（8社＋PFN）
- 出典一覧ページ（**109件の一次・二次ソースをカテゴリ別に整理**）
- FAQページ（本質的疑問への回答）

### 💡 JavaScript機能
- `js/main.js` — 共通ヘッダー／ドロワー／ボトムナビ／フッターの
  **動的マウント**、アニメーション、アコーディオン、
  カウンター、プログレスバー等の起動
- `js/data.js` — 全章メタデータ、年表、出資企業、出典109件、
  FAQなどを**一元管理するデータ層**

## 💾 データモデル / ストレージ

本サイトは**完全静的サイト**であり、サーバーサイドDBは使用していない。
すべてのコンテンツは以下に格納されている：

- **`js/data.js`** に一元化されたJSライフタイムオブジェクト
  - `SITE_DATA.chapters` — 全15章のメタ（id、slug、title、lead、tags）
  - `SITE_DATA.timeline` — 年表イベント群
  - `SITE_DATA.companies` — 出資・技術連携企業データ
  - `SITE_DATA.sources` — 出典109件（カテゴリ別）
  - `SITE_DATA.faq` — FAQ項目
- 各章ページ内の静的HTML（見出し・本文・引用）

RESTful Table APIは本プロジェクトの性格上**意図的に使用していない**。
